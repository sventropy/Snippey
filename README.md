# Snippey

iOS keyboard extension app for text based snippets

## Generate app icons

```
cd img
chmod +x genrate_icons.sh
./generate_icons.sh
```

## App Color Scheme

https://coolors.co/6dd48e-353535-ffffff-a0a0a0-042a2b

Colors:

- \#6DD48E --> Very light malachite green
- \#353535 --> Jet gray
- \#FFFFFF --> White
- \#A0A0A0 --> Quick Silver gray
- \#042A2B --> Dark green
